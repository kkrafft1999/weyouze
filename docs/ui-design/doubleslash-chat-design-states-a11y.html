a11y-Anpassungen vs. Original

color
Cyan #00A5E1 nur als Form (Border, Dot, Send-BG). Schrift & Pill-BG → Clickable Blue #00759E (5,0:1 Kontrast).
focus
2px Cyan Outline + 2px Offset auf allen interaktiven Elementen via :focus-visible.
html
<button> statt <div>, <input> statt <span>, <ol role="log"> für Konversation.
aria
aria-label auf Icon-Buttons, role="img" + Label auf Status-SVGs, aria-hidden auf dekorativen Zeichen.
motion
prefers-reduced-motion deaktiviert Pulse, Cursor-Blink, Spinner, Wave-Dots.
lang
"RUNNING", "DONE", "BAT AGENT" mit lang="en" — sonst spricht Screenreader sie deutsch.
size
Send-Button 32×32, Header-Icons mit 32×32 Touch-Target, Pill 11px statt 10px.
